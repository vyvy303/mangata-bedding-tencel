# Mangata Bedding - Premium Tencel Landing Page

A luxurious landing page for Mangata Bedding, featuring high-quality Tencel products, craftsmanship philosophy, and a serene aesthetic.

## Features
- **Premium Design:** Minimalist and elegant aesthetic with soft sage green tones.
- **Responsive:** Fully responsive layout built with Tailwind CSS.
- **Typography:** Uses Noto Serif and Manrope for a sophisticated feel.
- **Micro-interactions:** Smooth hover effects and transitions.

## Project Origin
This project was generated from a Stitch UI design and curated with custom high-quality imagery.
